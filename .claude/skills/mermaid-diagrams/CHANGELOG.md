# Changelog

All notable changes to the `mermaid-diagrams` skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-04-09

### Added
- `author: Ryan Smith` in `SKILL.md` frontmatter.
- `tags: [mermaid, diagrams, visualization, documentation]` in `SKILL.md` frontmatter for registry searchability.
- User-level and project-level installation instructions in `README.md`.

### Changed
- Distribution from private to public.
- `README.md` scope section updated to reflect public distribution.

## [1.0.0] - 2026-04-09

### Added
- Initial version of the `mermaid-diagrams` skill.
- `SKILL.md` with Autonomous Execution behavioral classification, trigger phrases, the core "use mermaid, not ASCII" rule, and a diagram-type selection guide.
- `references/diagram-types.md` explaining when to pick each mermaid type (flowchart, sequenceDiagram, stateDiagram-v2, erDiagram, classDiagram, gantt, gitGraph).
- `references/examples.md` with copy-pasteable examples for every supported diagram type.
- File/directory tree exception: `tree`-style ASCII is allowed only for filesystem layouts.
- `README.md` documenting scope (project-level, private), triggers, and contents.
